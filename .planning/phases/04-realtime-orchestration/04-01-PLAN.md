<objective>

**Superfície API tempo real Wave 1:** WebSocket multicasted (**Redis**) + **`DELETE`** cancelável + persistência **`celery_task_id`** para **revoke**.

Contrato fechado no **04-CONTEXT** (D‑04‑01 … D‑04‑14): `client_session` **mandatório**.

Out of wave: publisher worker detalhes de pipeline (**04‑02**); SPA (**04‑03**).

</objective>

<tasks>

### 04-01-01 — DB & modelo

- Alembic: colunas `jobs.celery_task_id VARCHAR` nullable + `jobs.status` estender valores permitidos com **`cancelled`** (migration + modelo SQLAlchemy API + modelo worker espelho).

### 04-01-02 — POST /jobs

- Após `send_task(...)`, gravar **`AsyncResult.id`** no campo `celery_task_id` no mesmo fluxo antes de responder (transação async).

### 04-01-03 — DELETE /api/v1/jobs/{id}

- Estado **pending/running**: setar cancel requested / `cancelled`; `celery_app.control.revoke(task_id, terminate=True)` se `celery_task_id` presente; publicar **`job.cancelled`** via Redis usando helper partilhado **mínimo** (ou mover publish inline aqui até `04-02` extrair pacote único).

- Estados terminais `success/failed/cancelled` ⇒ **409** body JSON.

### 04-01-04 — Gateway WebSocket

- `neuralcad_api/ws/job_channel.py`: `websocket /ws/jobs/{job_id}` (**Starlette WS**).

- Query **`client_session` obrigatório** (regex UUID).

- Lookup job existe ⇒ `accept`; enviar primeiro frame JSON `schemaVersion`:1 , `type`:`channel.ready` , inclui **`client_session` echoed**, `jobId`.

- Subscrição `redis.asyncio` pubsub `SUBSCRIBE neuralcad:job:{job_id}`, forward **text**.

- Fecha limpo upon disconnect.

### 04-01-05 — Observabilidade

- Log warning se DELETE sem task id (enqueue race).

### 04-01-06 — Tests

- `pytest`: DELETE transições; WS handshake negado sem `client_session`; com redis fake/mock forward one message injected.

### 04-01-07 — SUMMARY

- CURL exemplos DELETE + WS url.

</tasks>
